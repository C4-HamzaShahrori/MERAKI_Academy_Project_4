## 🏁 Getting Started <a name = "getting_started"></a>

clone Repostiory to your local machine

```
https://github.com/C4-HamzaShahrori/MERAKI_Academy_Project_4.git
```

in Backend gitbash
```
npm i 
npm i Mongo DB
```

in Frontend gitbash
```
npm i 
npm i axios
```

then in backend gitbash
```
npm run dev
```
and in frontend gitbash 
```
npm start
```



### Installing

if package json isn't installed:

```
npm init -y 
```

## 🎈 Usage <a name="usage"></a>

-POST request to /users route 
 
 input: 
 
 ```
 "firstName": String,
 "latName": String,
 "age": Number,
 "country" String,
 "email":String,
 "password": String
 
 ```
 
 output:
 
 ```
"success": true,
"message": "Success User Added"

 ```
